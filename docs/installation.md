# Installation

## Requirements

- Python 3.10+
- PyTorch 2.2+
- NumPy 1.26+
- Pillow 10+
- Matplotlib 3.10+

## Install

```powershell
python -m pip install -r requirements.txt
```

For conda:

```powershell
conda env create -f environment.yml
conda activate myenv
```

## Optional editable install

If you want to import the package from notebooks or ad hoc scripts:

```powershell
python -m pip install -e .
```
